Q：老师，除了逻辑回归，如果用神经网络的话，效果能不能更好
逻辑回归：线性的分类器，考虑的比较简单，如果规律简单，符合线性规律 => 适合逻辑回归，这时反而不需要用复杂的模型，比如神经网络
神经网络：非线性分类器，考虑的拟合比较复杂，如果规律复杂，符合非线性规律 => 可以用神经网络

Q：怎么把这个预测py，做成小模型

Q：为什么我跑出来的回归特征系数图片与老师的不一样
1）数据特征处理方法，可能不同

Q：codebuddy的模型文件是.pkl
模型文件，可以是 .pkl （存储是python变量） 需要通过import pickle来打开
也可以是 joblib，需要通过 joblib工具来打开

Q：什么叫超参，如何确定超参？超参和参数有什么区别？
超参数 => 人工可以调整的参数，在模型之外的
参数 => 模型内的参数，是模型训练的结果

y=wx + b 这里 (w,b)就是模型的参数
iteration = 10次，还是=15次，这个是人工指定的超参数

Q：生成了两个joblib模型文件，不是三个，正常吗

你的2个joblib，可能是 模型文件，特征列表

Q：怎么把两个gender变成一个系数？我这边是两个系数
两个gender，是因为做了特征的拆分，即 0-1编码。

不需要对特征值进行拆分（0-1编码），使用特征字段进行逻辑回归即可，比如gender字段，就直接使用gender字段放到逻辑回归中

需要对特征字段进行拆分（0-1编码），可以看到更细的颗粒度，帮我编写 logistic_regression_01.py

Q：问问两个方法有什么区别
1）决策树 => 非线性的交叉特征
各种特征交叉下的 决策，比如 30-60岁 & 婚姻已婚的 => 续保

2）逻辑回归 => 相对简单，对单独个特征的影响 看的比较清楚
特征之间是相互独立，比如 年龄大的 =>  更愿意续保
已婚的 => 更愿意续保
 
Q：树深设置成多少合适
通常3-7，建议可以先设置为4

Q：决策树是的深度过多，不是容易过拟合
是的，越复杂的模型 => 越容易过拟合

Q：怎么判断是过拟合了
训练集准确率: 0.9163
测试集准确率: 0.9100
=> 没有过拟合

训练集准确率: 0.9963
测试集准确率: 0.8100
=> 说明过拟合
